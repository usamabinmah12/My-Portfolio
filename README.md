# 🌐 Personal Portfolio Website

## 📖 Project Overview

This is my personal **portfolio website** built with React to showcase my skills, projects, and experience as a web developer. It features a modern, smooth, and interactive design focused on delivering a great user experience.

The portfolio highlights my work, technical skills, and provides an easy way for recruiters and collaborators to connect with me.

## 🔗 Live Demo

👉 https://usama-portfolio-henna.vercel.app

## ✨ Features

* 🧑‍💻 Modern and clean UI/UX design
* 📱 Fully responsive for all devices
* ⚡ Smooth animations and transitions
* 🎞️ Interactive motion effects using Framer Motion
* 🌀 Smooth scrolling experience using Lenis
* 📂 Projects showcase section
* 🛠️ Skills and technologies section
* 📞 Contact section for easy communication
* 🎨 Minimal and performance-focused design

## 🚀 Technologies Used

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Framer Motion (animations)
* Lenis (smooth scrolling)
* Tailwind CSS (if applicable)

## 🎯 Purpose

The purpose of this portfolio is to present my work in a clean and interactive way while demonstrating my frontend development skills, animation handling, and modern UI practices.
